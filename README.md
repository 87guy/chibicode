# メモ
paddingはborder（枠線）の内側の余白を作るに使う  
marginはborderの外側に余白を作るときに使う  
<input>要素は１行のテキスト入力を受け取るための要素、終了タグが不要  
<textarea>要素は複数行のテキスト入力を受け取るための要素です
